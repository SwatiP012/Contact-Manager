#  Contact Manager ‚Äì MERN Stack Application

Contact Manager is a **full-stack web application** that helps users manage their contacts efficiently.  
It is built using the **MERN stack (MongoDB, Express.js, React.js, Node.js)** and features a **modern UI, smooth animations, and real-time interactions**.

---

##  Features

-  Add new contacts
-  Edit existing contacts
-  Delete contacts
-  Search contacts instantly
-  Contacts displayed in alphabetical order
-  Contact types: Personal & Professional
-  Toast notifications for success actions
-  Modern UI with gradients and animations
-  Fully responsive design
